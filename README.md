# SudokuSolver
Building a sudoku solver in C.

Apporach:
Use known rules form sudoku to solve.
This approach cannot solve every sudoku as it needs a fallback (bruteforce) for hard problems, which I have not yet implemented.
Several rules are not (yet) implemented (Like x-wing, Swordfish,Jellyfish, etc.).
